# Week 4: Web Application Security - Part 1

Introduction to web application vulnerabilities and the OWASP Top 10.

## Learning Objectives
- Understand common web vulnerabilities
- Learn about SQL injection
- Practice XSS (Cross-Site Scripting) attacks
- Understand input validation importance

## Labs
1. **SQL Injection** - Understanding and exploiting SQL injection vulnerabilities
2. **Cross-Site Scripting (XSS)** - Identifying and preventing XSS attacks

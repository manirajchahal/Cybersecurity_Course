# Week 3: Cryptography Basics

## Overview
Introduction to cryptographic concepts and hands-on practice with encryption/decryption.

## Learning Objectives
- Understand symmetric and asymmetric encryption
- Learn about hashing algorithms
- Practice basic encryption techniques
- Understand SSL/TLS certificates

## Labs

2. **Hashing and Integrity** - Understanding hash functions and their applications
